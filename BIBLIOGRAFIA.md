# Bibliografía

## Bibliotecas y Frameworks Utilizados

### TensorFlow y Keras

- **TensorFlow**: Abadi, M., Agarwal, A., Barham, P., Brevdo, E., Chen, Z., Citro, C., ... & Zheng, X. (2016). TensorFlow: Large-scale machine learning on heterogeneous distributed systems. *arXiv preprint arXiv:1603.04467*.

- **Keras**: Chollet, F. (2015). Keras. GitHub repository. Disponible en: https://github.com/fchollet/keras

  - Documentación oficial: https://keras.io/
  - Chollet, F. (2018). *Deep Learning with Python*. Manning Publications.

### NumPy

- Harris, C. R., Millman, K. J., van der Walt, S. J., Gommers, R., Virtanen, P., Cournapeau, D., ... & Oliphant, T. E. (2020). Array programming with NumPy. *Nature*, 585(7825), 357-362. DOI: 10.1038/s41586-020-2649-2

  - Documentación oficial: https://numpy.org/doc/stable/

### Matplotlib

- Hunter, J. D. (2007). Matplotlib: A 2D graphics environment. *Computing in science & engineering*, 9(3), 90-95. DOI: 10.1109/MCSE.2007.55

  - Documentación oficial: https://matplotlib.org/

### scikit-learn

- Pedregosa, F., Varoquaux, G., Gramfort, A., Michel, V., Thirion, B., Grisel, O., ... & Duchesnay, E. (2011). Scikit-learn: Machine learning in Python. *Journal of machine learning research*, 12(Oct), 2825-2830.

  - Documentación oficial: https://scikit-learn.org/stable/
  - Citar scikit-learn: https://scikit-learn.org/stable/about.html#citing-scikit-learn

### Seaborn

- Waskom, M. L. (2021). seaborn: statistical data visualization. *Journal of Open Source Software*, 6(60), 3021. DOI: 10.21105/joss.03021

  - Documentación oficial: https://seaborn.pydata.org/

### Dataset CIFAR-10

- Krizhevsky, A., & Hinton, G. (2009). Learning multiple layers of features from tiny images. *Technical report, University of Toronto*. Disponible en: https://www.cs.toronto.edu/~kriz/learning-features-2009-TR.pdf

  - Dataset oficial: https://www.cs.toronto.edu/~kriz/cifar.html

### Perceptrones Multicapa (MLP)

- Rumelhart, D. E., Hinton, G. E., & Williams, R. J. (1986). Learning representations by back-propagating errors. *Nature*, 323(6088), 533-536. DOI: 10.1038/323533a0

- Goodfellow, I., Bengio, Y., & Courville, A. (2016). *Deep Learning*. MIT Press. Disponible en: https://www.deeplearningbook.org/

### Early Stopping

- Prechelt, L. (1998). Early stopping-but when?. *Neural Networks: Tricks of the trade*, 55-69. DOI: 10.1007/3-540-49430-8_3

### Optimizador Adam

- Kingma, D. P., & Ba, J. (2014). Adam: A method for stochastic optimization. *arXiv preprint arXiv:1412.6980*. DOI: 10.48550/arXiv.1412.6980
