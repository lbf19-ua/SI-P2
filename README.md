# Práctica 2: Visión artificial y aprendizaje

**Sistemas Inteligentes - Curso 2025/2026**  
**Universidad de Alicante**

## Descripción

Esta práctica implementa redes neuronales para clasificación de imágenes usando el dataset CIFAR-10. Incluye:

- **MLP (Perceptrones Multicapa)**: Implementación básica y optimización
- **CNN (Redes Neuronales Convolucionales)**: Implementación y optimización
- Visualización de resultados con Matplotlib
- Análisis de matrices de confusión

## Estructura del proyecto

- `practica2.py`: Código principal con todas las implementaciones
- `test_tensorflow_cuda.py`: Script para verificar TensorFlow y CUDA
- `verificar_gpu_host.sh`: Script para verificar GPU en el host

## Requisitos

- Python 3.8+
- TensorFlow/Keras
- NumPy
- Matplotlib
- scikit-learn
- seaborn

## Instalación

1. Crear entorno virtual:
```bash
python3 -m venv tensorflow
source tensorflow/bin/activate
```

2. Instalar dependencias:
```bash
pip install numpy matplotlib scikit-learn seaborn tensorflow
```

## Uso

```bash
source tensorflow/bin/activate
python3 practica2.py
```
